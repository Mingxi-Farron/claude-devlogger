# Generate Final Project Assessment

Read all session logs from `devlog/logs/` in chronological order.

Read `devlog/TIME.md` for total time invested.

Read `devlog/README.md` for project info.

Generate a comprehensive final assessment and save to `devlog/FINAL_ASSESSMENT.md`:

```markdown
# Final Project Assessment: [PROJECT_NAME]

## Project Overview
- **Description:** [from README]
- **Tech Stack:** [from README]
- **Duration:** [start date] to [end date]
- **Total Time:** [from TIME.md]
- **Sessions:** [count]

## Timeline
Brief chronological summary of major phases/milestones from each session.

## Key Challenges & Solutions
Aggregate all problems from session logs:
- What were the biggest challenges?
- What solutions worked?
- What approaches failed and why?

## Lessons Learned
- Technical insights gained
- Process improvements discovered
- What would you do differently?

## Final Outcome
- What was accomplished?
- How does it compare to original goals?

## Reflections
Long-term perspective on the project journey.
```

Ask the user:
1. "Any additional reflections or lessons learned to include?"
2. "What would you do differently if starting over?"

Incorporate their answers and save the file.

Confirm by showing the file path and a brief summary.
