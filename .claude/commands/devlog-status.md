# Devlog Status

Read `devlog/TIME.md` to get:
- Total project time
- Number of sessions
- Most recent session info

Check `devlog/logs/` for the most recent session log.

If a session is currently active (today's log exists without a Duration in header):
- Show "Active Session" status
- Session number
- Start time
- Goals for this session
- Time elapsed so far

Show project summary:
- Total sessions completed
- Total time invested
- Last 3 session summaries (date, duration, focus)

If no active session:
- Suggest starting one with `/devlog-start`
